== Description ==

Mailmaga-X is a simple and powerful plugin help for sending mailmagazines to your customers.

Features of Mailmaga-X:

1. Export and Import user list by CSV file.
2. Manager mail magazine users with group.
3. Support multi Mail Server to send large number of mail magazines at the same time.
4. Send mail magazines in background so don't have to wait.
5. Calculate Email Open Rate Automatically (Only support HTML mail).

== Installation ==

1. Upload `mailmaga-x` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setting your Mail Server in `Mailmaga-X/Servers` menu.
4. Import your mail magazine users from csv file.

== Frequently Asked Questions ==

= What about Interval in Server setting? =

Interval could be used for controlling Mail Sending Interval, in some SMTP Mail Server, mail will be blocked when your site send mail too fast .
